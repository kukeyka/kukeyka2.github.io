Использование атрибута style для стиля элементов HTML
Использование background-color для цвета фона
Использование color для цвета текста
Использование font-family для шрифта текста
Использование font-size для размера текста
Использование text-align для выравнивания текста
<b> - Жирный текст
<strong> - Важный текст
<i> - текст Курсив (наклонный)
<em> - текст Акцентирует внимание (наклонный)
<mark> - Маркерованный текст
<small> - Небольшой текст
<del> - Удаленный (зачеркнутый) текст
<ins> - Вставленный (добавить) текст
<sub> - Подстрочный текст
<sup> - Надстрочный текст

Тег	Описание
<b>	Определяет жирный текст
<em>	Определяет выделенный текст
<i>	Определяет часть текста в альтернативном голосе или настроении
<small>	Определяет меньший текст
<strong>	Определяет важный текст
<sub>	Определяет подстрочный текст
<sup>	Определяет надстрочный текст
<ins>	Определяет вставленный текст
<del>	Определяет удаленный текст
<mark>	Определяет маркированный/выделенный текст
